🚀 Auto UI ↔ API Checker
📌 Overview
A testing tool that automatically interacts with the UI, extracts displayed data, and validates it against API responses to ensure data consistency.

🎯 Key Features
🔄 Automate UI interactions (click, input, navigation)
📊 Extract data directly from UI
🔗 Fetch and parse API responses
⚖️ Compare UI data with API data
🚨 Highlight mismatches and inconsistencies
