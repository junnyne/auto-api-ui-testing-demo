# 🚀 Auto UI ↔ API Checker

## 📌 Overview

A lightweight testing tool that automatically interacts with the UI, extracts displayed data, and validates it against API responses to ensure consistency and accuracy between frontend and backend.

---

## 🎯 Key Features

* 🔄 Automates UI interactions (clicks, inputs, navigation)
* 📊 Extracts dynamic data directly from the UI
* 🔗 Fetches and parses API responses
* ⚖️ Compares UI data with API data
* 🚨 Highlights mismatches and inconsistencies
